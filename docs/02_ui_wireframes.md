# Bastion UA — UI Wireframes (опис екранів + логіка)

Формат: ASCII-вайрфрейм + опис інтерактивних елементів. Готовий до передачі UI/UX дизайнеру для Figma-переведення.

**Орієнтація**: portrait (мобілка первинна), 9:19.5 (iPhone 14+), landscape на PC/Steam опціонально.

**Ієрархія екранів**:
```
[Splash] → [Auth] → [Tutorial (first-time)] → [HUB: Map]
                                               ├─ Front Sector (бій)
                                               ├─ Tech Tree
                                               ├─ MFA (дипломатія)
                                               ├─ HQ / Ставка (prestige)
                                               ├─ Units
                                               ├─ Events log
                                               ├─ Historical Archive
                                               ├─ Shop / Battle Pass
                                               └─ Settings / Підтримати партнерів
```

## S-03. Front Sector (tower-defense екран)

```
┌──────────────────────────────┐
│ ← Київ · Північний сектор ⏸ ⏩ │
├──────────────────────────────┤
│  Ворог:  ████░░░░  65%       │
│  Ми:     ███████░  86%       │
├──────────────────────────────┤
│                              │
│  [Бій 5×5 grid]              │
│                              │
│   🏰R5 ─ 🛡️R4 ─ ⚔️R3 ─ 🪖R2 ─ 🎯R1 │
│    │    │    │    │    │    │
│    │    │    │    │    │    │
│   BR1  BR2   BR3  BR4   BR5  │
│                              │
│  ← ворог атакує з R1→R5 →    │
│                              │
├──────────────────────────────┤
│ 🎁 Слоти підрозділів         │
│ [ Vanguard ] [ Falcons ] [+ ]│
│ [ Shield   ] [   +     ] [+ ]│
├──────────────────────────────┤
│ 💥 Бусти:                    │
│ [FPV-рій cd] [Арта cd] [Mavic]│
├──────────────────────────────┤
│      [ ✓ ПОЧАТИ БІЙ ]        │
└──────────────────────────────┘
```

**Інтерактив**:
- Кожна зона (R1..R5) = слот для підрозділу; drag-and-drop з нижньої панелі
- 5 слотів = 5 підрозділів одночасно
- Ліва панель — доступні підрозділи (грід з кнопкою "+" = купити / розблокувати)
- Бусти: 3 активних, кулдауни
- Після "Почати бій" — auto-battle у прискоренні, гравець може лише скасовувати бусти
- Результат: win/lose/partial → звіт + нагорода
- Pause/Fast-forward (2×, 4×)

**Варіанти сектору**:
- Urban (Київ/Харків): бонус ТрО +30% DEF
- Forest (Чернігів): засідки, +20% crit для ДРГ
- Steppe (Херсон): відкритий фронт, бонус артилерії
- Ice (зима): уповільнення танків

## Стиль

**Палітра**:
- Primary: `#005BBB` (синій прапор)
- Accent: `#FFD500` (жовтий прапор)
- Success: `#2E7D32` (зелений — свобода)
- Danger: `#D32F2F` (червоний — бій)
- Warning: `#F57C00` (жовто-помаранчевий — загроза)
- Neutral: `#1A1A2E` (темно-синій фон), `#E8EAF6` (світлий)

**Шрифти**:
- Headline: **e-Ukraine Head** (офіційний шрифт держави, доступний безкоштовно)
- Body: **e-Ukraine** або **Inter**
- Monospace (для цифр/таймерів): **JetBrains Mono**

**Іконографія**:
- Власний набір стилізованих символів; державна чи бригадна символіка використовується лише після окремого rights clearance
- Heroicons / Lucide — базові UI-іконки
- Без Lucide.russian.svg (жарт, але принципово лише UA-сумісне)

**Анімації**:
- DOTween для tween'ів
- Lottie для складних (сирени, тривога)
- 60 fps min
- Haptic feedback на кожен tap

**Accessibility**:
- Контраст WCAG AA
- Режим дальтоніка (protanope/deuteranope)
- Розмір шрифту: регулюється у налаштуваннях
- Voice-over для iOS, TalkBack для Android
