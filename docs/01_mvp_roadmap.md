# Bastion UA — MVP Roadmap (6 місяців)

## Ціль MVP
Запустити **закриту бету** на Android + Telegram Mini App з:
- Оборона Києва (24.02 – 02.04.2022) як єдина повна кампанія
- 3 напрямки фронту (Київ-північ, Чернігів, Суми)
- 10 стилізованих підрозділів, 15 типів ворожих юнітів
- Idle-прогресія + 1 цикл prestige
- Екран підтримки партнерських фондів через зовнішні посилання
- Аналітика, crash-reporting, telemetry

**Ціль бети**: 5 000 DAU, D1 retention ≥ 40%, D7 ≥ 15%, session length ≥ 8 хв.

## Тижневий план (26 тижнів)

### Місяць 1. Foundation (W1–W4)

**W1 — Kick-off**
- [ ] Формалізація бренду: назва, лого, палітра
- [ ] GDD v0.1 (Game Design Document)
- [ ] Git repo (GitLab/GitHub), Trello/Linear борд, Slack/Discord
- [ ] Git-flow, CI (GitHub Actions), артефакти на Nexus
- [ ] Unity проєкт (2022 LTS), Addressables, URP
- [ ] Addressables pipeline для ассетів
- [ ] Backend skeleton: FastAPI + PostgreSQL + Alembic + Sentry

**W2 — Core loop prototype**
- [ ] Idle-таймер: Entropy-подібний ресурс (Набої)
- [ ] Click-to-produce + auto-tick (Tick 1 c)
- [ ] JSON-конфіг юнітів
- [ ] Серверний save/load (anonymous device_id)

**W3 — Карта v0**
- [ ] Статична карта України (low-poly / ілюстрація)
- [ ] 3 області (Київ, Чернігів, Суми) як клікабельні регіони
- [ ] Стан області: свобода / під загрозою / окуповано
- [ ] Відображення фронту (Bezier-лінія між точками)

**W4 — Auto-battle ядро**
- [ ] Сектор = 5 слотів юнітів, до 5 ліній
- [ ] Базова логіка: HP, DMG, range, speed, cooldown
- [ ] Результат бою = зміна стану сектору
- [ ] 3 базові юніти (піхота, БМП, ДРГ) з обох сторін

### Місяць 2. Content + Progression (W5–W8)

**W5 — Дерево апгрейдів (Tech Tree)**
- [ ] 20 вузлів на старті (5 гілок × 4 рівні)
- [ ] UI дерева (SVG-подібний layout)
- [ ] Розблокування через Ideas

**W6 — 10 стилізованих підрозділів**
- [ ] Моделі/спрайти (Synty для low-poly прев'ю)
- [ ] Стати: 10 стилізованих підрозділів, натхненних ролями оборони Київщини 2022
- [ ] Кожна — унікальний перк
- [ ] Система рангів (1–10), досвід

**W7 — Друга валюта: Ideas / Технології**
- [ ] Розблокується після "Стоп Гостомель" (історичний трігер)
- [ ] Дерево технологій (25 вузлів)
- [ ] Research-таймер (офлайн-прогрес)

**W8 — Дрони та артилерія**
- [ ] Mavic-розвідка як активний скіл на карті
- [ ] FPV-рій як AoE-атака (cd 5 хв)
- [ ] Bayraktar як бос-кілер (обмежений запас, дорого)
- [ ] Балансна таблиця v1

### Місяць 3. Polish core + Events (W9–W12)

**W9 — Історичні івенти**
- [ ] Подієвий engine: JSON-сценарії (trigger → dialog → choice → reward)
- [ ] 5 івентів: "Гостомель", "Ірпінь-Буча", "Чорнобаївка", "Затоплення Ірпеня", "Крилата ракета на Київ"
- [ ] Лок-рішення: 2–3 варіанти дії, різні нагороди
- [ ] Історичний контекст без реальних цитат, likeness або прав на конкретні матеріали

**W10 — Prestige / Ставка**
- [ ] Умова: звільнити 3 області → "Київська Перемога"
- [ ] Ресет симуляції, отримання **Badges**
- [ ] Ставка: 15 permanent-апгрейдів
- [ ] Лог перемог у Historical Archive / Hall of Honor

**W11 — Моральний дух + мобілізація**
- [ ] Окрема валюта, капається з перемог
- [ ] Витрачається на мобілізацію підрозділів
- [ ] Штрафи при втратах міст (паніка)

**W12 — Дипломатія**
- [ ] Окремий екран "МЗС України"
- [ ] 5 партнерів: США, UK, Польща, Німеччина, ЄС-фонди
- [ ] Quests → unlock ленд-ліз (Javelin, Stinger, NLAW)

### Місяць 4. UX/UI Polish (W13–W16)

**W13 — UI/UX final pass**
- [ ] Редизайн з фіналізованим брендом
- [ ] Анімації (DOTween)
- [ ] Haptic-feedback для тапів (мобілка)
- [ ] Accessibility: контраст, розмір шрифту, дальтонік-режим

**W14 — Звук**
- [ ] Ambient: "Київ на світанку", "Сирени", "Гуркіт"
- [ ] SFX: постріли, вибухи, радіо-перемовини
- [ ] Український диктор озвучує повідомлення ("Увага! Повітряна тривога!")
- [ ] Музика: патріотичний інструментал (не нав'язливо, casual-ніш)
- [ ] Mute кнопка завжди доступна

**W15 — Tutorial + onboarding**
- [ ] 5-хвилинний guided tour
- [ ] Soft-tutorial через стилізованого NPC-гіда штабу
- [ ] First-time UX: перша перемога за 90 с

**W16 — Hall of Honor / Historical Archive**
- [ ] Нейтральний екран історичного контексту кампанії
- [ ] Без реальних імен, портретів, біографій чи персональних меморіальних матеріалів у MVP
- [ ] Без геймплей-ефекту, лише контекст і атмосфера
- [ ] Посилання на офіційні ресурси можна додати після окремого legal review

### Місяць 5. Монетизація + Бек (W17–W20)

**W17 — Backend production-ready**
- [ ] JWT auth (Apple Sign In + Google + guest)
- [ ] Базові профілі та лідерборди без чатів/UGC
- [ ] Rate-limits, anti-cheat (server-side валідація)
- [ ] PostgreSQL + pgvector для нього не треба, лише PostgreSQL + Redis
- [ ] Daily backup → S3-compatible (Hetzner / Wasabi)

**W18 — IAP інтеграція**
- [ ] Google Play Billing, App Store IAP
- [ ] Telegram Stars (для TG Mini App)
- [ ] Пакети: Cubes ($1.99/$4.99/$9.99/$19.99), Battle Pass ($9.99)
- [ ] Infinity Pass (без реклами) $4.99/міс

**W19 — Донат-інтеграція**
- [ ] Пункт меню: "Підтримати партнерські фонди"
- [ ] Зовнішні deep-link на партнерські фонди
- [ ] Без винагород, валюти або power-бонусів за зовнішній донат
- [ ] Комунікація про social mission і monthly reports
- [ ] Юридичне оформлення — окремий договір (див. 05_legal.md)

**W20 — Реклама (opt-in, rewarded)**
- [ ] AppLovin MAX або Unity LevelPlay

- [ ] Без banner/interstitial — тільки опціональні
- [ ] Вимкнення через Infinity Pass
