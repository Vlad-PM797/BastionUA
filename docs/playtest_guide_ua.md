# Bastion UA — Інструкція для тестера (5–20 хв)

Дякуємо, що тестуєш **ранній прототип** idle-strategy про оборону України. Це не фінальна гра — нам важлива **чесність** і перші враження.

---

## Як запустити

### Windows
1. Розпакуй `BastionUA-Windows-demo.zip`
2. Запусти `BastionUA.exe` (папка `BastionUA_Data` має бути поруч)
3. Якщо SmartScreen попереджає — «Додаткова інформація» → «Все одно запустити» (unsigned dev build)

### Android (якщо отримав APK)
1. Дозволи встановлення з невідомих джерел
2. Встанови `BastionUA-dev.apk`
3. Landscape або portrait — обидва працюють, але UI заточений під широкий екран

---

## Що робити (15 хв сесія)

1. **Прочитай onboarding** — не пропускай першу підказку
2. **Тапай по карті / кнопці** — збирай **Набої** (ammo)
3. **Обери регіон** (Київ, Чернігів, Суми, Харків) — подивись статус
4. **Натисни Battle** — проведи бій за окупований/небезпечний регіон
5. **Пройди story event**, коли з’явиться popup (Гostomel → … → Kharkiv)
6. **Купи upgrade**, якщо вистачить ресурсів
7. Спробуй дійти до підказки **Prestige** (звільни всі окуповані регіони + 4 events)

### Гарячі клавіші (Windows, optional)
| Клавіша | Дія |
|---------|-----|
| T | Tap (ammo) |
| 1–4 / QWER | Регіони |
| B | Battle |
| S | Save |
| P | Prestige (коли доступно) |

## На що звернути увагу

- Чи зрозуміло **що робити далі**? (objective hint зверху/збоку)
- Чи **читається текст** на кнопках і в popup?
- Чи **зрозумілий battle** — перемога/поразка, витрата ammo?
- Чи **цікаві events** — хочеться читати чи skip?
- Чи **хочеться продовжити** після 10 хв?

## Відомі обмеження прототипу

- Звук — прості synthetic beeps, не фінальний audio
- 4 регіони, 4 events — не повна кампанія MVP
- Немає monetization, backend, ads
- Save локальний (один пристрій)
