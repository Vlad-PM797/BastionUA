# Bastion UA — Voice-Over Script v3 (UK)

Статус: draft  
Дата: 2026-04-24  
Мова: українська  
Формат: voice-over для рекламно-оглядового ролику  
Цільова тривалість: ~2:55-3:10

---

## 1. Performance Notes

### Подача

- спокійно;
- впевнено;
- без надриву;
- без “дикторського пафосу”;
- з короткими паузами після сильних фраз.

### Темп

- середній;
- не поспішати в першій третині;
- трохи прискорити в середині;
- фінал знову зробити повільнішим і впевненішим.

### Інтонація

Не “реклама продукту за будь-яку ціну”, а:

**“Ось проєкт, який має форму, зміст і потенціал.”**

## 5. Recording Notes

Перед записом:
- прибрати надто довгі паузи між реченнями;
- перевірити вимову `idle-strategy`, `soft-power`, `Bastion UA`;
- зробити 2 версії:
  - більш стриману;
  - трохи енергійнішу для середини ролика.

Після запису:
- підрізати дихання, але не прибирати всю “людяність”;
- залишити природні мікропаузи;
- не стискати голос надто сильно компресором.
